Important FAQ
What is the Document Object Model(DOM)?

The DOM is a programming interface for HTML and XML documents, representing the page's structure as a tree of objects that can be manipulated with JavaScript.


How do you select an element from the DOM?

We can select an element using methods like document.querySelector(), document.getElementById(), document.getElementsByClassName(), or document.getElementsByTagName().


What is event delegation in the DOM and why is it useful?

Event delegation is a technique where a parent element listens for events on its child elements by utilizing event bubbling, reducing the need for multiple event listeners and improving performance.


How do you manipulate an element's attributes and styles using the DOM?

We can use setAttribute(), getAttribute(), or removeAttribute() for attributes and modify the style property directly (e.g., element.style.color = "red";).
